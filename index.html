LKT JS

LKT JS presents a set of tools designed to improve the development of Javascript / Typescript apps.

Features:

  • Framework independent
  • Dependency free
  • Lightweight
  • Typescript typing hint
  • Install via NPM

These packages are used as dependencies by LKT Vue project.

If you want it, you can take a look at the organization page
